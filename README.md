# simuladordeinvestimentos

## Descrição
```
<h2> Descrição </h2>
<p> Projeto proposto pela empresa EQI Investimentos como teste para candidatura à uma vaga de Front End Developer Jr. </p>
<p> O projeto consiste em retornar os dados da API, ao clicar no botão simular, preenchendo os campos com os valores recebidos, e gerando o gráfico também de acordo com os valores recebidos da API. </p>
<a href="https://github.com/eqi-investimentos/desafio-frontend"></a>
<p> A proposta para desenvolver o projeto em React porém eu desenvolvi em VueJs que é o Framework que conheço no momento. </p>
<p> Para realização, foi consumida uma API fornecida pela empresa EQI Investimentos. </p>
<p> <a href="https://github.com/eqi-investimentos/desafio-fake-api"></a> </p>
<p> No layout foi utilizado CSS puro e Bootstrap. </p>
<p> Para gerar o gráfico, foi utilizado Vue Chartjs. </p>
```
